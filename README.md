# FCSC Command Center

A full-stack event management system designed for the Faculty of Computing Student Community (FCSC), enabling efficient planning, task tracking, and team collaboration.

---

## 🚀 Features

- Event creation and management
- Task assignment and progress tracking
- Role-based access (EC Member / General Member)
- Event-wise team collaboration
- Dashboard with real-time insights

---

## 🧠 Tech Stack

### Frontend
- React (Vite)
- Tailwind CSS
- React Router

### Backend
- Supabase (PostgreSQL, REST APIs)

---

## 🗂️ Database Design

- Events
- Tasks
- Users (Profiles)
- Event Members (Many-to-Many)
- Announcements
